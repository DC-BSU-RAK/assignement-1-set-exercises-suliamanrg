HTML Quick Referance

10 HTML tags

  • html-Defines the root element of an HTML document.
  • head-Contains meta-information about the document, such as the title and links to stylesheets.
  • body-Contains the content of the HTML document that is visible to users.
  • title-Sets the title of the web page, which is displayed in the title bar or tab of the browser.
  • h1,h2,h3,etc-Define header sections with different levels of importance (h1 is most important).
  • p-Represents a paragraph of text.
  • a-Creates a hyperlink to a different web page or resource.
  • img-Inserts an image onto the web page.
  • ul-Defines an unordered (bulleted) list.
  • ol-Defines an ordered (numbered) list.

CSS Quick Referance

08 CSS tags

  • color-sets the text color.
  • background-color-Sets the background color of an element.
  • font-sizeDefines the font size
  • margin-Adds space outside an element
  • paddingCreates space within an element, between borders and its content.
  • border- Creates a border around an elemen
  • text-align- Specifies the horizontal text alignment in an element.
  • display- Controls how an element is displayed (e.g., block, inline, flex, etc.).
